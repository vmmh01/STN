Cada carpeta señala un defecto de Tela y contiene 
imagenes de las muestras de tela con el defecto. 

Además, contiene una subcarpeta llamada "recortes" 
donde se puede encontrar la zona especifica del 
defecto sobre las imagenes originales.

Estas últimas, estan en escala de grises y en formato 
PNG para que puedan ser leídas por el sistema de detección de defectos.
